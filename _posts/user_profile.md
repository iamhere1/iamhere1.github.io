# 用户画像是什么

   用户画像是根据用户的社会属性及各类行为，抽象出一个标签化的用户模型。其核心工作主要包括两点：
   
**构建标签集:** 根据实际业务需求、平台、数据等，确定构建用户画像的标签集合。

**为用户贴上标签:** 根据用户的社会属性和各类行为数据，利用机器学习模型及相关规则，为用户贴上标签集合中的标签。
